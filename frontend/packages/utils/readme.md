# Utils

Ce package contient les fonctions utilitaires partagées entre toutes les applications frontend.

## Rôle

- Formatage de dates, nombres, textes
- Fonctions de validation de formulaires
- Helpers divers (debounce, throttle, etc.)
- Fonctions génériques non liées à l’UI ou à l’API

## Objectif

Éviter la duplication de code et garder des utilitaires communs accessibles
à toutes les applications frontend.
