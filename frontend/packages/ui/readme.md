# UI

Ce package contient les composants UI partagés entre toutes les applications frontend.

## Rôle

- Boutons, modales, inputs, formulaires, tableaux, etc.
- Layouts réutilisables
- Composants de feedback (loader, alertes, notifications)

## Objectif

- Garantir une cohérence visuelle entre les différentes apps
- Réduire la duplication de composants
- Faciliter la maintenance et l’évolution du design
