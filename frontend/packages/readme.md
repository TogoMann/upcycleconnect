# Packages partagés

Ce dossier contient le code partagé entre toutes les applications frontend.

## Contenu

- `api-client/` : Fonctions pour appeler l’API backend (fetch/axios)
- `ui/` : Composants UI réutilisables (boutons, modales, formulaires, etc.)
- `auth/` : Gestion de l’authentification, des rôles et des permissions
- `utils/` : Helpers (formatage dates, validation, etc.)

## Objectif

- Éviter la duplication de code
- Garantir une cohérence entre les différentes applications
- Faciliter la maintenance
