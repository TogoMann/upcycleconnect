# Applications frontend

Ce dossier contient les différentes applications Vue.js selon le type d’utilisateur :

- `public-web/` : Espace Particulier
- `pro-web/` : Espace Professionnel / Artisan
- `staff-web/` : Espace Salarié
- `backoffice/` : Espace Administration

Chaque application :
- A son propre router
- Ses propres pages et layouts
- Utilise les packages partagés pour l’API, l’auth et l’UI
