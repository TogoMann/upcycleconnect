**Espace Utilisateurs**

L'accent est mis sur l'engagement et la facilité d'utilisation. Le système de dépôt via conteneur avec code est le point technique critique ici (flux : validation -> envoi de code -> ouverture physique ou logique).

- Engagement : Upcycling Score et tutoriel interactif (overlays).
- Transactions : Marketplace (dons/ventes) et réservation de services/formations.
