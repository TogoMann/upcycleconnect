# Utils

Ce dossier contient les fonctions utilitaires partagées dans tout le backend.

## Rôle

- Fonctions de formatage (dates, strings, etc.)
- Helpers de validation
- Fonctions communes (hash de mot de passe, génération d’ID, etc.)
- Outils génériques non liés à un module métier précis

## Objectif

Éviter la duplication de code et centraliser les fonctions génériques réutilisables.
